## Development
Create a `.env` file containing the following variables:
```dotenv
AUTH_SECRET

# OAuth
AUTH_GITHUB_ID
AUTH_GITHUB_SECRET

MONGODB_URI
RAPID_API_KEY
```

`AUTH_SECRET` can be generated using `npx auth`.
